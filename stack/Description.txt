Program: Stack Implementation with Infix to Postfix Conversion and Evaluation

This program demonstrates stack operations using a class-based implementation in C++.
The project is divided into three files:
1. Stack.h   → Header file containing class definition and function prototypes
2. stack.cpp → Implementation file containing function definitions
3. test.cpp  → Driver program with menu-driven interface to test stack operations

Features Implemented:
1. Basic Stack Operations:
   - Push
   - Pop
   - Display
   - Peek (Top element)
   - Check if stack is full
   - Check if stack is empty

2. Expression Handling:
   - Infix to Postfix Conversion:
     Converts a given infix expression (e.g., A+B*C) into postfix form (ABC*+).
     Uses stack to handle operator precedence and parentheses.
   - Postfix Evaluation:
     Evaluates a postfix expression (e.g., 23*4+) to produce the final numeric result.
     Supports operators: +, -, *, /, ^ (power).
     Includes error handling for invalid input and division by zero.

Constraints:
- Maximum stack size: 10 elements (defined by MAX constant)
- Uses array-based stack implementation
- Input validation included (stack overflow/underflow, invalid operators)
