Program: Menu-Driven Circular Singly Linked List Operations

This program demonstrates basic operations on a circular singly linked list using a menu-driven approach:
1. Create a circular singly linked list (nodes point back to head)
2. Display the list (shows elements in circular form ending with "Back to Head")
3. Insert a node at the start
4. Insert a node at the end
5. Delete a node from the start
6. Delete a node from the end

Constraints:
- Dynamic memory allocation used for nodes
- Handles empty list cases and single-node list cases
- Maintains circular property after every operation

Sample Run:
