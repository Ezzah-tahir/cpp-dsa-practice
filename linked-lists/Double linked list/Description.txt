Program: Menu-Driven Doubly Linked List Operations

This program demonstrates basic operations on a doubly linked list using a menu-driven approach:
1. Create a doubly linked list (nodes connected with both next and prev pointers)
2. Display the list in forward direction
3. Insert a node:
   - At the start
   - At the end
   - At a specific location
4. Delete a node:
   - At the start
   - At the end
   - At a specific location

Constraints:
- Dynamic memory allocation used for nodes
- Handles empty list cases and single-node list cases
- Maintains proper forward and backward links after every operation
