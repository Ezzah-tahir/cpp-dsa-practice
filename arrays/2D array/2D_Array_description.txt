Program: Menu-Driven 2D Array Operations

This program demonstrates basic operations on a 2D array using a menu-driven approach:
1. Array creation with user-defined rows and columns
2. Displaying elements in matrix form
3. Insertion of a new row (at start, at end, or at a specific position)
4. Deletion of a row (at start, at end, or at a specific position)
5. Searching for an element (returns row and column position)
6. Calculating the average of all elements

Constraints:
- Maximum size: 30 rows × 30 columns
- Input validation included (invalid positions, exceeding max size)
